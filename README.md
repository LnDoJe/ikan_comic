# 前言
 之前爬风之动漫漫画太少就放弃了
 后爬的漫画柜提供的网页
 花了整整三天研究漫画柜页面的逻辑..
 终于艰苦的爬下来了
 ## 技术栈
 * 前端框架:vue+vuex+axios+vue-pull-to+vue-router-elemen.ui 
 * 后端 : express+pm2
 * 数据库:mysql(dataBase:kankan)
 * 手机端调试:eruda
